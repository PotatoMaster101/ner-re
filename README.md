# NER-RE
Playing around with NER and RE.

## Setup
### Windows
```shell
python -m venv .venv
./.venv/Scripts/Activate.ps1
pip install -r requirements.txt
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu130
```

### Unix
```shell
python -m venv .venv
. ./.venv/bin/activate
pip install -r requirements.txt
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu130
```
